## 起動コマンド

```bash
# 開発時
$ npm run start

# ウォッチモード
$ npm run start:dev

# プロダクションモード
$ npm run start:prod
```

## テスト

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# プロダクション版の実行手順

```bash
# Next.jsのビルド
$ npm run build:next

# Nest.jsのビルド
$ npm run build

# ビルド成果物の実行
$ npm run start:prod
```
